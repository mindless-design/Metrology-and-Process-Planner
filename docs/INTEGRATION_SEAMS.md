# Integration Seams

Last updated: 2026-06-25

## Audit Position

`SessionDocument` and `SessionRecord` are the canonical saved-state spine. UI models, KLayout
overlays, process outputs, report files, CSV files, SVG/PNG drawings, and PowerPoint/PDF outputs
are derived views or artifacts with provenance recorded back into the document.

## Audit Result

The core seams are explicit and mostly test-backed. The highest-risk remaining partial seam is live
KLayout rebinding for saved overlays after opening a document without an already-bound layout. The
dependency audit found and fixed one workflow-to-app command import bypass during this pass.
